# FC6-HighPingKick-Patch

How to install:

Download Injector.exe and Patch.dll and place into GameFolder\bin or bin_plus depending on which FarCry6.exe is running (example: C:\Program Files (x86)\Ubisoft\Ubisoft Game Launcher\games\Far Cry 6\bin_plus, where FarCry6.exe is located.)

Launch the game, go into singleplayer first then run the Injector.exe as admin. You'll then be greeted by a box telling you that the patch was successful.
Then have your partner join and enjoy finally being able to play together :)

(Only the host needs to have the patch injected.)
